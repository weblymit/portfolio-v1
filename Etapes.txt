NEXTJS 
////// 
1- Install contentful = npm i contentful
2- Connexion à l'API contentful dans settings->API keys->Add APIkeys
3- Créer un fichier .env et verifier dans gitignore que c'est ignoré
4- Dans .env créer les variables d'environnement en uppercase préfixé par NEXT_PUPLIC
5- Coller les variables d'environnement récuperer dans contentful
6- Dupliquer .env.local et renommer en .env-exemple pour aider la personne qi retravaillera sur mlon projet
